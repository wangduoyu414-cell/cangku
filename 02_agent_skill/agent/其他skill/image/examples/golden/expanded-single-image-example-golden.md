## Task Summary

为 Instagram reel 封面整理一份完整提示词包，重点是改造前后差异、标题安全区、缩略图可识别和创作者原生感。

## Main Prompt

Delivery Goal: Instagram reel cover.

Create a vertical cover image for a storage makeover reel where the before-and-after contrast is obvious at thumbnail size. Reserve a stable title-safe zone, keep the key subject inside the vertical-safe center band, and use a simple mobile-readable hierarchy. The overall tone should feel creator-native rather than like paid-media creative.

## Hard Constraint Block

- Must Keep:
  - before-and-after difference is obvious at a glance
- Must Not Appear:
  - ad-like performance marketing tone
  - title area cropped by platform UI
- Safe Zone:
  - reserve a clean cover-title-safe-zone
  - keep the key subject inside the vertical-safe-zone

## Soft Preference Block

- creator-like realism
- native platform feel
- simple, strong mobile hierarchy

## Parameter Suggestions

- aspect ratio: 9:16 vertical cover
- quality: medium first to validate thumbnail readability, then high for final render
- iterations: likely 2 passes for crop safety and preview clarity

## Risk Notes

- thumbnail loses the before-after contrast
- title area collides with crop or UI
- layout looks too polished and ad-like on mobile

## Iteration Patches

- Keep the same layout, but enlarge the core contrast area if the thumbnail reads weakly.
- Keep the same tone, but reduce graphic polish if the result feels too much like paid media.

## Acceptance Checklist

- thumbnail recognizability is strong
- title stays inside the safe zone
- mobile readability is clear
- the cover feels creator-native
