# Agent + Skill 指南包

本目录当前的 `Skill` 设计主入口为：

- `SKILL_STANDARD_UNIFIED.md`

## 使用建议

- 想设计或升级 `Skill`：先读 `SKILL_STANDARD_UNIFIED.md`
- 想看 `skill.yaml` 机器字段：再读 `SKILL_MANIFEST_SPEC.md`
- 想看 `evals` 设计与门禁：再读 `SKILL_EVALS_SPEC.md`
- 想看模型升级检查与删减规则：再读 `SKILL_UPGRADE_CHECKLIST.md`
- 想做平台对接、注册中心、编排器、治理系统：再读 `RUNTIME_CONTRACT_SPEC.md`

## 文档分工

- `SKILL_STANDARD_UNIFIED.md`：统一设计目标、边界、不变量、生命周期和执行计划
- `SKILL_MANIFEST_SPEC.md`：`skill.yaml` 字段规范
- `SKILL_EVALS_SPEC.md`：正式评测目录、最小覆盖和升级评测流程
- `SKILL_UPGRADE_CHECKLIST.md`：升级前、中、后的检查清单
- `RUNTIME_CONTRACT_SPEC.md`：平台运行时公共契约
