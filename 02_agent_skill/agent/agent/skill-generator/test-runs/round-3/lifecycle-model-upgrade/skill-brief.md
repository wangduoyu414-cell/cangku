# multilingual-risk-triage brief（技能简报）

## Goal（目标）
- After a stronger model upgrade, simplify the multilingual risk triage skill without weakening audit, rollback, or baseline rules.

## Scope（边界）
- Simplify triage scaffolding after model improvement.

## Non-Scope（非边界）
- Do not remove validation or baseline requirements.

## Inputs（输入）
- model upgrade note（模型升级说明）

## Outputs（输出）
- simplified triage artifacts（精简后的分诊产物）
- model upgrade note（升模说明）

## Risks（风险）
- Too much simplification may erase escalation nuance.

## Success Bar（成功门槛）
- Must list removed scaffolding and new misuse risks.
