# federated-incident-governor brief（技能简报）

## Goal（目标）
- Upgrade the federated incident-governor skill after long-form failures, stale outputs, and cross-round regression drift.

## Scope（边界）
- Upgrade the governor skill from regression and stale-output failures.

## Non-Scope（非边界）
- Do not turn the skill into a generic incident platform.

## Inputs（输入）
- cross-round regressions（跨轮回归）
- stale output reports（陈旧产物报告）

## Outputs（输出）
- iterated governor artifacts（迭代后的治理产物）
- failure attribution（失败归因）

## Risks（风险）
- Fixing stale outputs may hide useful historical comparisons.

## Success Bar（成功门槛）
- Must explain how regression and stale output failures are addressed.
